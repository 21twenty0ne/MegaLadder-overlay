## Инструкция по установке

Этот скрипт добавляет продвинутый оверлей статистики для игры на [ladder.megabonk.su](https://ladder.megabonk.su/).
Оверлей поддерживает режим **Picture-in-Picture (PiP)**, что позволяет закрепить окно со статистикой поверх окна самой игры.

### Шаг 1: Установка расширения
Установите расширение **Tampermonkey** для вашего браузера:
- [Tampermonkey для Chrome / Edge](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Tampermonkey для Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)

### Шаг 2: Добавление скрипта
1. Нажмите на иконку Tampermonkey в браузере и выберите **"Создать новый скрипт"** (Create a new script).
2. Удалите весь стандартный код из открывшегося окна.
3. Скопируйте содержимое файла `megaladder overlay.user.js` и вставьте его в редактор.
4. Нажмите **Файл -> Сохранить** (File -> Save).

### Шаг 3: Использование
1. Откройте страницу лобби или матча на [ladder.megabonk.su](https://ladder.megabonk.su/).
2. В верхней части экрана появятся кнопки управления:
    - **АКТИВЕН / ОТКРЫТЬ**: Управление режимом "Картинка в картинке" (PiP).
    - **ВИДИМОСТЬ**: Настройка отображаемых элементов (аватары, билды, баны).
    - **НАСТРОЙКИ**: Выбор языка, масштаба текста и иконок.

## Installation Guide

This script adds an advanced stats overlay for [ladder.megabonk.su](https://ladder.megabonk.su/).
The overlay supports **Picture-in-Picture (PiP)** mode, allowing you to place the stats window directly over your game screen.

### Step 1: Install Extension
Install the **Tampermonkey** extension for your browser:
- [Tampermonkey for Chrome / Edge](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Tampermonkey for Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)

### Step 2: Add Script
1. Click the Tampermonkey icon in your browser and select **"Create a new script"**.
2. Delete all default code in the editor window.
3. Copy the content of `megaladder overlay.user.js` and paste it into the editor.
4. Click **File -> Save**.

### Step 3: Usage
1. Open any lobby or match page on [ladder.megabonk.su](https://ladder.megabonk.su/).
2. Control buttons will appear at the top:
    - **ACTIVE / OPEN**: Toggle Picture-in-Picture (PiP) mode.
    - **VISIBILITY**: Customize visible elements (avatars, builds, bans).
    - **SETTINGS**: Change language, text scale, and icon sizes.
